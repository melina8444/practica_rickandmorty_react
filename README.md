# Rick & Morty React App

Aplicación web desarrollada con React que consume la API pública de Rick & Morty para mostrar personajes de la serie.

🔗 Live Demo  
https://practica-rickandmorty.netlify.app/

---

## 🚀 Características

- Consumo de API externa
- Renderizado dinámico de personajes
- Manejo de estado con React
- Interfaz simple y responsive

---

## 🛠 Tecnologías utilizadas

- React
- JavaScript
- HTML5
- CSS3
- REST API

---

## 📡 API utilizada

Rick and Morty API  
https://rickandmortyapi.com/
